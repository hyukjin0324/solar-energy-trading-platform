# solar-energy-trading-platform

AI-driven solar power generation prediction combined with a simulated peer-to-peer (P2P) energy trading platform. The system predicts how much solar power a household will generate based on weather data, then matches surplus and deficit households for energy trading with dynamic pricing.

## Project background

Solar power adoption is growing rapidly in India. This project aims to enable households with solar panels to predict their generation and trade surplus electricity with neighbors, reducing dependence on the centralized power grid.

## Objectives

- Predict solar power generation using weather and PV sensor data
- Match households with surplus energy to households with a deficit
- Apply dynamic pricing based on supply and demand
- Simulate automated settlement between buyers and sellers

## Scope

This is a simulation-based project. Real-world grid integration and payment gateway implementation are outside the scope.
